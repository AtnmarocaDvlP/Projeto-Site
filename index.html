Balancr

Projeto acadêmico de 4 meses que esta sendo desenvolvido desde o terceiro periodo da Faculdade, desenvolvido em equipe. O Balancr é um aplicativo de controle financeiro pessoal, onde os usuários podem registrar suas receitas e despesas, visualizar gráficos de gastos e gerar relatórios mensais.


O projeto foi desenvolvido utilizando a linguagem Dart e C++ com o framework Flutter, permitindo a criação de uma interface amigável e responsiva. Nos trabalhamos em conjunto para implementar funcionalidades como categorização de transações, alertas de gastos excessivos e exportação de dados.

todo o codigo fonte e os processos estão registrados no github, segue o link do repositório: Balancr
